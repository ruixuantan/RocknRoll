# Rock n Roll
A die rolling application.

## Running the project manually
### Server
```shell
sbt rocknrollServer/run
```

### Web
```shell
cd rocknroll-web
npm run start
```

## Ports
| Services| Port Number |
| ------- | ----------- |
| Server  | 8080        |
| Postgres| 5432        |
| Web     | 4200        |
